Bug tracker

Bugy, ktoré sme opravili

🐍
_canvas vs _mpl_canvas konflikt
CTkFrame má interný self._canvas. NetworkDiagramView ho prepisoval. Fix: premenovať na self._mpl_canvas.
🔤
Slovenské znaky → čierne štvorce v PDF
Helvetica (Type1) nepodporuje Latin Extended-A. Fix: DejaVu TTF z matplotlib bundle cez reportlab TTFont.
✂️
Názov projektu orezaný vertikálne v PDF
VALIGN=MIDDLE + auto row height klipovalo text. Fix: 2-row layout so SPAN + VALIGN=TOP.
🎨
color_blend(bg=BG_CARD) zlyháva
BG_CARD je tuple, nie string. Fix: color_blend_dual() helper — generuje (light, dark) naraz.
📐
pack vs grid TclError na Dashboarde
chart_frame používal grid, ale matplotlib canvas bol embedovaný cez pack(). Fix: grid(row=1) + grid_rowconfigure.
💥
AttributeError: content_area
Keď Dashboard crashol, self.content_area sa nenastavil. Každý klik na sidebar → crash. Fix: inicializácia v __init__ + getattr check.
