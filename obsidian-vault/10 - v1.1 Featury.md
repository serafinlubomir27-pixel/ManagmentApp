## Gantt Chart — scroll pre veľké projekty

**Súbor:** `ui/screens/gantt_chart.py`

Pre projekty s **>12 úloh** alebo **šírka figúry >= 18 palcov** sa chart embeduje do `tk.Canvas` scrollovateľného viewportu namiesto priameho grid embedu.

### Štruktúra

```
GanttChartView (CTkFrame)
└── tk.Canvas (viewport)      ← scrollovateľné
│   └── tk.Frame (inner)
│       └── FigureCanvasTkAgg (matplotlib)
├── CTkScrollbar (vertical)
└── CTkScrollbar (horizontal)
```

- Mouse wheel scroll (Windows): `viewport.yview_scroll(-delta/120, "units")`
- `<Configure>` bind na inner frame → aktualizuje `scrollregion`
- Threshold: `SCROLL_THRESHOLD = 12`

## Deadline notifikácie

**Súbor:** `ui/screens/dashboard.py`

Banner pod headerom, zobrazuje sa len ak existujú blížiace sa deadliny.

### Logika

```python
today = datetime.date.today()
days_left = (due - today).days
if days_left < 0:    → "overdue" (červená)
elif days_left == 0: → "dnes!" (červená)
elif days_left == 1: → "zajtra" (oranžová)
elif days_left <= 3: → "za N dni" (žltá)
```

- Zobrazí max 5 alertov, potom "+ N ďalších"
- Filtruje completed úlohy
- Zdroj: `task_repo.get_tasks_with_due_dates(user_id)` — assigned_to OR project owner
- Banner sa **nezobrazí** ak nie sú žiadne upcoming deadliny

## Admin — celý strom hierarchie

**Súbory:** `logic/hierarchy.py`, `ui/screens/team_view.py`

### `get_full_tree(manager_id) -> list[dict]`

BFS prechod celého stromu podriadených, max hĺbka 10 (ochrana voči cyklom v dátach).

Každý člen dostane naviac:
- `depth: int` — 0 = priamy, 1 = podriadeý priameho, ...
- `manager_name: str` — meno priameho nadriadeného

### UI — toggle button

V `TeamView` má admin tlačidlo **"Celý strom"**:
- Aktívny: PRIMARY farba, zobrazí rekurzívny strom
- Neaktívny: ghost štýl, zobrazí len priamych podriadených (default)

Karty členov sa vizuálne indentnú podľa `depth` (20px per úroveň) a zobrazujú `↳ Podriadeý <manager>`.

## CSV Export / Import

**Súbory:** `logic/export_manager.py`, `ui/screens/project_detail.py`

### Export (`CSV ↓` tlačidlo)

```python
export_tasks_csv(project_id, file_path) -> (bool, str)
```

- UTF-8 BOM (`utf-8-sig`) — správne otváranie v Excel
- Hlavičky po slovensky (napr. "Názov", "Trvanie (dni)")
- Polia: id, name, status, priority, category, duration, due_date, assigned_username, ES/EF/LS/LF/float, is_critical, description, notes

### Import (`CSV ↑` tlačidlo)

```python
import_tasks_csv(project_id, file_path, created_by) -> (bool, str)
```

- Akceptuje BOM aj bez BOM
- Mapuje slovenské hlavičky späť na field names
- Validuje: `name` povinné, `status`/`priority` whitelist
- Per-row error reporting (max 5 + "... a N ďalších")
- Po importe: `_recalc_cpm()` + refresh task listu

### CSV formát (príklad)

```
ID,Názov,Stav,Priorita,Kategória,Trvanie (dni),...
1,Analýza požiadaviek,completed,high,Planning,5,...
2,Návrh databázy,in_progress,medium,Design,3,...
```
