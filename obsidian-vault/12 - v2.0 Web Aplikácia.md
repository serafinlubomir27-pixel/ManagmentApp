## Backend (FastAPI)

### Autentifikácia
- JWT tokeny (python-jose), 24h platnosť
- Rolový systém: `admin` | `manager` | `employee`
- `GET /auth/login` (form data) → `{ access_token, user_id, username, role }`
- Dependency injection: `get_current_user()` vo všetkých chránených endpointoch

### Hlavné routery

| Router | Prefix | Popis |
|--------|--------|-------|
| `auth_router` | `/auth` | Login, register, JWT |
| `projects_router` | `/projects` | CRUD + CPM trigger |
| `tasks_router` | `/tasks` | CRUD úloh, time log, komentáre |
| `team_router` | `/team` | Správa používateľov, hierarchia |
| `attachments_router` | — | Prílohy projektov a úloh |
| `clients_router` | `/clients` | Klientský modul (Fáza 3) |

### Databáza
SQLite s `database/setup.py` — automatické migrácie pri štarte:
```python
# Príklad migrácie (idempotentná):
cursor.execute("ALTER TABLE task_attachments ADD COLUMN visibility TEXT DEFAULT 'team'")
```

### StaticFiles
Nahrané súbory servírované cez FastAPI:
```python
app.mount("/uploads", StaticFiles(directory="uploads"), name="uploads")
```

## Frontend (React + TypeScript)

### Stack
- **React 18** + **TypeScript**
- **Tailwind CSS** — utility-first štýly
- **React Query** — server state, cache, mutácie
- **Axios** — HTTP klient (`frontend/src/api/client.ts`)
- **Vite** — build tool + dev server
- **React Router v6** — SPA routing

### Kľúčové komponenty

| Komponent | Popis |
|-----------|-------|
| `NetworkDiagram.tsx` | SVG sieťový diagram s klikateľnými uzlami |
| `TaskDetailModal.tsx` | Dvojsĺpcový modal — info + komentáre + prílohy |
| `AttachmentSidebar.tsx` | Panel príloh v ProjectDetailPage |
| `FileUploadDropzone.tsx` | Drag & drop upload so selector viditeľnosti |
| `AttachmentList.tsx` | Zoznam súborov s delete + zmena viditeľnosti |
| `PipelineKanban.tsx` | Kanban pre deal pipeline klienta |
| `ComplianceChecklist.tsx` | MiFID II compliance checklist |
| `MeetingLog.tsx` | Log stretnutí s klientom |

## Deployment

### Railway (Backend)
- Auto-deploy z `main` branch na GitHub
- Env vars: `SECRET_KEY`, `CORS_ORIGINS` (ak treba extra domény)
- `Procfile` alebo `railway.json` riadi štart

### Surge.sh (Frontend)
- Manuálny deploy príkazom: `npx surge dist managmentapp.surge.sh`
- `vercel.json` obsahuje SPA rewrite pravidlá (aj pre Surge funguje cez `index.html` fallback)
- Certifikát platí 29 dní (automaticky obnovovaný)

### CORS konfigurácia (backend/main.py)
```python
_allowed_origins = [
    "http://localhost:5173",
    "https://managmentapp.surge.sh",
    "https://managmentapps.netlify.app",
] + [o.strip() for o in os.environ.get("CORS_ORIGINS","").split(",") if o.strip()]
```
